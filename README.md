# RAG

This project is a simple Retrieval-Augmented Generation (RAG) application built in Python. It appears to use a document ingestion and indexing pipeline, along with a vector store, to retrieve relevant information before generating answers. The repository includes files such as `app.py`, `index_document.py`, and `calculators.py`, along with `doc/` and `vector_store/` folders, which suggests a workflow for loading documents, storing embeddings, and answering queries more accurately from retrieved context.
